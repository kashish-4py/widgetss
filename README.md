# 🐾 My Cute Desktop Companion

A lightweight, borderless desktop widget featuring a cute animated pixel cat that plays a sound when clicked! Built using Python and PyQt6.

## ✨ Features
- 🌸 Floating, borderless design.
- 🐈 Smooth GIF animation support.
- 🎵 Interactive sound effect on click.
- 📌 Drag and drop anywhere on your screen.

## 🚀 How to Run
1. Install dependencies: `pip install PyQt6`
2. Run `catwidget.pyw`
3. **Right-click** anywhere on the widget to close it!
